# 人工客服功能实现说明

## 功能概述

已成功在 `ProfileView.vue` 个人中心页面实现了人工客服功能，用户点击"人工客服"按钮后会自动生成加密的客服链接并在新窗口中打开。

## 实现步骤

### 1. 安装依赖

- 安装了 `jsencrypt` 库用于RSA加密
- 安装了 `crypto-js` 作为备用
- 安装了相应的TypeScript类型定义

### 2. 创建RSA加密工具函数 (`src/utils/rsa.ts`)

- 实现了 `rsaEncrypt()` 函数，使用提供的公钥进行RSA加密
- 实现了 `generateCustomerServiceUrl()` 函数，生成完整的客服链接

### 3. 修改ProfileView.vue

- 导入RSA工具函数
- 添加 `goToCustomerService()` 点击处理函数
- 为"人工客服"按钮添加点击事件

## 加密流程

1. **构建基础URL**: `https://kf.keff88.vip/index/index/home`
2. **构建查询参数**:
   ```
   visiter_id=&visiter_name=&avatar=&business_id=17&groupid=0&special=24&user_id={当前用户ID}
   ```
3. **添加随机密码**: 在查询字符串末尾添加 `&pwd={随机字符串}`
4. **RSA加密**: 使用提供的公钥对完整查询字符串进行RSA加密
5. **生成最终URL**: `https://kf.keff88.vip/index/index/home?code={加密后的数据}`

## 使用说明

1. 用户登录后进入个人中心页面
2. 点击"人工客服"按钮
3. 系统自动：
   - 获取当前用户ID
   - 生成加密的客服链接
   - 在新窗口中打开客服页面
4. 如果用户未登录，会提示先登录

## 错误处理

- 用户未登录时会显示提示信息
- RSA加密失败时会显示错误提示
- 添加了详细的控制台日志用于调试

## 测试方法

1. 启动开发服务器: `npm run dev`
2. 访问 http://localhost:3000
3. 登录用户账号
4. 进入个人中心页面
5. 点击"人工客服"按钮
6. 查看浏览器控制台的加密过程日志
7. 确认新窗口打开了正确的客服链接

## 注意事项

- 客服链接会在新窗口中打开，确保浏览器允许弹窗
- 加密过程在浏览器客户端完成，确保用户数据安全
- 生成的链接包含用户身份信息，用于客服系统识别用户
